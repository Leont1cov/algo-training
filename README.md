# Algorithms & JS Core

Репозиторий посвящен изучению Computer Science и углубленному освоению JavaScript через решение алгоритмических задач.

## Содержание
- **Grokking Algorithms**: Реализация алгоритмов из книги А. Бхаргавы на JS.
- **LeetCode Solutions**: Решения задач с фокусом на оптимизацию и чистоту кода.
- **JS Deep Dive**: Разбор особенностей языка (Event Loop, Prototype, Closures) в контексте задач.

## Технологии
- **Language**: JavaScript (ES6+)
- **Environment**: WebStorm / Node.js
